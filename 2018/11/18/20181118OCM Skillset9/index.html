Skillset 9

TASKS:

1. Install RAC

1.1. Install RAC, known as RACDB, the installation media is placed in the /stage/database directory

ORACLE_BASE=/u01/app/oracle
ORACLE_HOME=/u01/app/oracle/product/11.2.0/dbhome_1

1.2. The database using the DATA disk group

1.3. Using the General purpose method to create the database, do not configure EM

1.4. The use of Policy-managed mode management database, Service pool name is OCMPOOL

1.5. Flashback area using FRA disk group, size: 4G

1.6. Memory target needs to be set to 800M

2. The configuration of RAC

2.1 Create Sequence, requirements according to sequence generated sequence, in order to improve the performance of all values should be placed in memory, the start value: 100, the end value: 99999

2.2 Create a Service:ACNTPAY,Requirements for connection to OCMPOOL, each time using only one instance.
