Skillset 7

TASKS:

1. Create Physical Standby Database.

1.1 Primary Database is PROD1,Standby Database is SBDB1;The Standby database to be created in Management Server

1.2 Standby database data and log files should be placed in the directory that /u01/app/oracle/oradata/SBDB1.

2. Management Physical Standby Database.

2.1 The standby database is in the open state, execute the script(vdedu1:/home/oracle/scripts/sbdb1.sql).

3.Restore Physical Standby Database

3.1 Into standby database state from the open state

3.2 Let the standby database synchronization can be maintained timely with Primary database.

3.3 Let the Primary database can quickly complete the recovery process (opening the block change tracking)

3.4 Archive log of Primary database not in preparing a database application cannot be deleted. (the primary database’s archive log transfer all standby database, then delete)

4.Configure Physical standby Database

4.1 Create the Service, known as SBDB1SRV, when EMREP is used as a standby database, this service can be started automatically.

4.2 Increasing the PRODSRV local Service in the Database Server, so that when the primary database is broken, can Failover to prepare for Standby database.
